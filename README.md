# Dragon Skins Mod

Custom skins for Dragon Client - Works with Fabric, Forge, and Quilt!

## Features

- ✅ Load custom skins from Dragon Client launcher
- ✅ Works with all Minecraft versions (1.16.5 - 1.21.1)
- ✅ Supports Fabric, Forge, and Quilt
- ✅ Automatic fallback to default skins
- ✅ No configuration needed

## Installation

### Automatic (Recommended)
The Dragon Client launcher automatically installs this mod when you upload a custom skin.

### Manual
1. Download the mod for your Minecraft version from [Releases](https://github.com/dragon-client/dragon-skins-mod/releases)
2. Put it in your `mods` folder
3. Launch Minecraft

## How It Works

1. Dragon Client launcher runs a local skin server on `localhost:25585`
2. This mod intercepts Minecraft's skin loading
3. Checks the local server for custom skins first
4. Falls back to Mojang's servers if no custom skin exists

## Building

```bash
cd dragon-skins-mod
./build.sh
```

Requires:
- Java 17+
- Gradle 8.x
